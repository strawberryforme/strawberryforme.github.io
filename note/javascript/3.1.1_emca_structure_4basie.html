    # 1、string

    属性：
    str.length
构造函数方法：
    String(thing) 
    String.fromCharCode(num1[, ...[, numN]]) 返回：一个长度为 N 的字符串
    String.raw`Hi\n${2+3}!`;
原型方法：(4+3+4+3=14个)
    +
    str.concat(str2, [, ...strN]) 返回：一个新的字符串，包含参数所提供的连接字符串。强烈建议使用赋值操作符 (en-US)（+, +=）代替 concat 方法
    str.repeat(count)
    str.padEnd,str.padStart(targetLength, padString) 返回：新的填充字符串

    str.slice(beginIndex[, endIndex]) 返回一个从原字符串中提取出来的新字符串
    str.substring(indexStart[, indexEnd])包含给定字符串的指定部分的新字符串。
    str.trim() 字符串两端去掉空白的新字符串。

    str.charAt,str.charCodeAt(index) 返回：指定的字符
    str.indexof,str.lastIndexOf(searchString, position)返回:第一次索引。或者返回-1
    str.includes(searchString[, position]) 返回 true,或则false。
    str.endsWith,str.startsWith(searchString[, length]) 返回：长度内最后是以...结尾的。

    str.split([separator[, limit]]) 返回源字符串以分隔符出现位置分隔而成的一个 Array
    str.toLowerCase() str.toUpperCase()
    str.valueOf()
继承：
            Function+Object

    Number.isNaN(value) 返回：要检测是否为 NaN 的值。
    Number.parseFloat(string) 返回：一个字符串解析成浮点数。
    Number.parseInt(string, radix) 返回：依据指定基数，解析字符串并返回一个整数。
