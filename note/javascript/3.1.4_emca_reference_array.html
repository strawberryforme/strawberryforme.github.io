    类：对象+length属性。例如 arguments对象,NodeList,HTMLCollection,DOM对象。
    在它们上调用数组方法，即使它们本身没有这些方法。
    类数组转为数组：
        1.Array.from(arrayLike); 
        2.var args = [...arguments];
            [Symbol.iterator]: Array.prototype[Symbol.iterator]。需要添加iterator接口。
        3.Array.prototype.slice.call(alo); 此方法如果不传参数的话会返回原数组的一个拷贝
        Array.prototype.concat.apply([], arrayLike)
        ①Array.prototype.slice，这段是执行slice，slice不传参数其实就是克隆数组。
        ②让slice方法的this指向类数组（arrayLike或arguments）
        实现：其实就是用call改变this指向，this变成谁，就会循环谁，把它里面每一项赋给新数组。
        4.for (var i = 0; i < cArr.length; i++) {arr.push(cArr[i]);}
